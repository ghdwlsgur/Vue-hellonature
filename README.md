## Vue 라우터를 사용하여 html 페이지 렌더링하기
- 본 프로젝트는 특별 기능없이 html 페이지를 뷰 라우터 기능을 사용하여 렌더링 한 것입니다.


## 실행방법
```javascript
npm i (노드 모듈 설치)
npm run dev (포트번호 8080)
```
